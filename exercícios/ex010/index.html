Usando links externos

links

Para usar link é usado a tag "a" que significa Âncora;

Com essa tag é armazenado os links.

link externo

Ideal para colocar links de sites que não são seus;

É usado as tags target="_blank" e rel="external" dentro da tag "a";

No link abaixo o link abrirá uma nova janela não fechando sua pagina:

  • Você pode acessar o meu repositório público no GitHub

link normal

No exemplo a baixo o link vai ser carregado por cima da sua pagina:

  • Você também pode acessar o meu canal no YouTube

Usando links internos

O link interno vem de uma outra página guardada na mesma pasta. Ultiliza-se a tag a para chamar o link. No link ctrl+space para ver as pastas internas que podem ser chamadas.
