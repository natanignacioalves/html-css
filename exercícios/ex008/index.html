Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

E no outro paragrafo, temos outro texto marcado no final.

Texto Grande e Pequeno

Temos um texto grande e um texto Pequeno nesse parágrafo usando as tag BIG e SMALL.

Texto Deletado

Podemos marcar texto como excluído para que indicar que ele deve ser lido, mas não considerado. Ultiliza-se a tag DEL

Texto Inserido

Podemos também indicar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois. Ultiliza-se a tag INS (semântica).

Existe também o sublinhado com a tag U (não semântica).

Texto Sobrescrito

Para inserir coisas do tipo x20+3.

Ultiliza-se a tag SUP.

Texto Subscrito

Para inserir coisas do tipo H2O.

Ultiliza-se a tag SUB.
