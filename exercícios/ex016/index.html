Testando Gradiente

Degradê de cores

Faz-se um Degradê ultilizando a tag background-image: linear-gradient().

Dentro dos parenteses a primeira coisa a se colocar é para onde sera direcionado o Degradê. No caso deste site está to top que seria direcionado para cima, porém você pode direciona-lo tanto para Right, bottom, lefth, ou até mesmo colocar o ângulo desejado.

exemplo da configuração do site: background-image: linear-gradient(to top, #012340, #025939, #027333, #03A63C, #04D939 ); background-attachment: fixed;
