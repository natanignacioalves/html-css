Listas Ordenadas

Usa-se a tag ol que signifca Listas Ordenadas.

Dentro da tag vai os itens da lista, representado por outra tag li para cada ítem da lista:

  1. Acordar
  2. Tomar café
  3. Escovar os dentes

Dentro da tag ol, pode se colocar um comando padrão "type" que lhe dará 5 opcões:

opção 1

Númerada normal:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Opção A

Lista alfabética com letra maiúscula:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Opção a

Lista alfabética com letra minúscula:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Opção I

Númerada por números romanos maiúsculos:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Opção i

Númerada por números romanos minúsculos:

  1. Acordar
  2. Escovar os dentes
  3. Tomar café

Listas não ordenadas

Como o próprio nome sujere, são listas que não precisam ser enumeradas ou sem a necessidade de uma sequência.

Em listas ordenadas a tag éra ol(Order Lists), em listas não ordenadas a tag é ul(Unordered List).

  • Pão
  • Leite
  • Tomate
  • Feijão

Nestas listas temos apenas 3 tipos de padrão type: disc, circle e square

Esses padrões apenas irão mudar a forma das bolinhas de cada item

Lista de Definições

Para a lista de definições(Definition list) é usada a tag dl.

É colocado um termo e a definição do mesmo. Como um dicíonario.

É usado a tag dt para o termo e dd para definição

Termo
Definição ou descrição

exemplo:

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do desing de um site.
Javascript
Linguagem de programação para a criação de interatividade de um site.
